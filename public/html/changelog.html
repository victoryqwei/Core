Core - Changelog and Suggestion:

Version 1.0.0:

  • Drawing player to screen.
  • Added player class.
  • Centered player on screen.
  • Added grid.
  • View other multiplayer players.

Version 1.1.0:

  • Individual name tags and colors.
  • Added cannon
  • Fixed grid scaling

Version 1.2.0:

  • Added player interpolation for smoother performance
  • UI elements added:
  • Added minimap to corner for broader view angle
  • Cooldown timer for gun
  • Added hotbar item slots
  • Fixed name tags and repositioned to top of player display
  • Other people's cannons can now be viewed by other players.

Version 1.3.0:

  • Added new tile elements:
  • Core
  • Wall
  • Added projectile interpolation for smoother performance
  • Added map boundaries
  • Hit detection for players and projectiles (respawned randomly)
  • Randomized spawning for players and cores

Version 1.4.0:

  • Added placeable tile elements with highlight
  • Core
  • Walls
  • Added placing tiles by other players
  • Added map border
  • Added hotbar controls and images
  • Cleaned code / created new core class and tile class
  • Added three core levels
  • Added core halo effect

Version 1.5.0:

  • Fixed UI elements
  • Fixed minimap core scale projection
  • Added better haloes around higher level cores
  • Respawn at core is present, else reload
  • Hit detection for client walls and player’s walls

Version 1.6.0:

  • Fixed bug that made gameplay extremely laggy when placing core with alot of wall tiles.
  • Added player collision with walls (currently, you don’t collide with your own walls)
  • Added tips bar above the hotbar
  • Added core power, each level core generates (core level) power units every second
  • Walls now cost one core power to create.
  • Base UI for cores added.
  • Player now has an access range for how far they can place walls and their core
  • Added angle interpolation
  • Fixed bug that cause invalid colors for the player

Version 1.7.0:

  • Walls and core cannot be placed outside of map
  • Shooting from barrel instead of center of player
  • Fixed issue if player tabs out of window the player stops sending and receiving data
  • Respawn timer after dying
  • Added leaderboard
  • Added input element to receive name instead of using prompt
  • Players are automatically disconnected if they do not send data for more than 3 seconds
